# 链式代理

## 配置示例

```yaml
Proxy Groups:
# 代理链，目前 relay 可以支持 udp 的只有 vmess/vless/trojan/ss/ssr/tuic
# wireguard目前不支持在relay中使用，请使用 proxy 中的 dialer-proxy 配置项
# Traffic: Clash <-> http <-> vmess <-> ss1 <-> ss2 <-> Internet
- name: "relay"
  type: relay
  proxies:
    - http
    - vmess
    - ss1
    - ss2
```

流量去向为 Clash <-> http <-> vmess <-> ss1 <-> ss2 <-> Internet
